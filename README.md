## 📌 Overview

StudyBuddy AI is an intelligent learning companion designed to help students overcome one of the biggest challenges in self-learning: inconsistency.

Unlike traditional study planners, StudyBuddy AI remembers user preferences, weak subjects, progress history, and career goals to provide personalized recommendations across multiple sessions.

The more the student interacts with StudyBuddy AI, the better the recommendations become.

---

## ✨ Key Features

### 🧠 Persistent Memory Engine

* Remembers student preferences
* Stores career goals
* Tracks learning interests
* Maintains progress history

### 📚 Personalized Study Planner

* Generates daily study schedules
* Creates weekly learning plans
* Adjusts recommendations based on progress

### 🎯 Weak Subject Analyzer

* Identifies weak areas
* Prioritizes important topics
* Suggests improvement strategies

### 🚀 AI Career Roadmap Generator

Creates learning roadmaps for career paths such as:

* Full Stack Developer
* AI Engineer
* Software Engineer
* Cybersecurity Analyst
* Data Analyst

### 💡 Smart Project Recommendations

Suggests beginner-to-advanced projects based on skills and interests.

Examples:

* Calculator App
* Weather Dashboard
* Portfolio Website
* Expense Tracker
* AI Chatbot

### 📈 Progress Tracking Dashboard

Monitor:

* Daily goals
* Weekly progress
* Learning streaks
* Completed tasks

### 🔔 Intelligent Reminders

Provides recommendations such as:

* Revision reminders
* Practice suggestions
* Project deadlines

### 📝 Session History

Allows students to revisit previous recommendations and continue learning without losing progress.
